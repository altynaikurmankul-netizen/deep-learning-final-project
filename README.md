# Airline Sentiment Analysis Using Deep Learning

## Project Description

This project focuses on sentiment classification of airline-related tweets using deep learning methods.
The goal is to classify tweets into positive, neutral, or negative sentiment categories.

## Dataset
Dataset: Twitter US Airline Sentiment Dataset
Source:
https://www.kaggle.com/datasets/crowdflower/twitter-airline-sentiment

## Planned Models
Baseline: TF-IDF + Logistic Regression
Deep Learning Model: LSTM Neural Network

## Evaluation Metrics
Accuracy
Precision
Recall
F1-score
Confusion Matrix
